# 사용 가이드

## 빠른 시작

### 1. 프로젝트 빌드 및 실행

```powershell
# 프로젝트 빌드
node build.js

# 애플리케이션 실행
npm start
```

또는 개발 모드로 실행:

```powershell
npm run dev
```

### 2. 배포용 실행 파일 생성

```powershell
npm run dist
```

실행 파일은 `release` 폴더에 생성됩니다.

## 주요 기능 사용법

### 데이터 입력

1. **행 추가**: 우측 상단의 "+ 행 추가" 버튼 클릭
2. **설명 입력**: 여러 줄로 작성 가능, 자동으로 높이 조절됨
3. **날짜 입력**: YYYY.MM.DD 형식 (예: 2024.01.01) 또는 자동 포맷 (20240101 입력 후 Enter)
4. **비중 입력**: 0~100 사이의 값 입력
5. **점수 입력**: 0~100 사이의 값 입력
6. **자동 계산**: 예상평가점수가 자동으로 계산됨 (비중 × 점수 / 100)

### 실시간 계산

- 비중이나 점수를 입력하면 **즉시** 예상평가점수가 계산됩니다
- 모든 행의 합계가 자동으로 업데이트됩니다
- 비중 합계가 100%인지 자동으로 확인됩니다 (100%이면 녹색, 아니면 빨간색)

### 데이터 저장

**방법 1: Save (Ctrl+S)**
1. File 메뉴 → Save 클릭 (또는 Ctrl+S)
2. 현재 파일에 저장됨 (모든 탭 데이터 포함)
3. 처음 저장하는 경우 Save As가 실행됨

**방법 2: Save As (Ctrl+Shift+S)**
1. File 메뉴 → Save As 클릭 (또는 Ctrl+Shift+S)
2. 저장할 위치와 파일명 선택
3. 파일은 `.dat` 확장자로 저장됨 (Base64 인코딩)
4. 모든 탭의 데이터가 함께 저장됩니다

### 데이터 불러오기

**Load (Ctrl+O)**
1. File 메뉴 → Load 클릭 (또는 Ctrl+O)
2. 저장된 `.dat` 파일 선택
3. 모든 탭과 데이터가 자동으로 복원됨
4. 타이틀 바에 파일명 표시
5. 구 버전 파일도 자동으로 변환하여 로드

### 행 삭제

1. 삭제할 행의 체크박스 선택
2. "× 행 삭제" 버튼 클릭
3. 선택된 모든 행이 삭제됨

## 계산 공식

```
예상평가점수 = 비중 × 점수 ÷ 100
예상평가점수 합계 = Σ(각 행의 예상평가점수)

이전평가점수 = 이전 비중 × 이전 점수 ÷ 100
이전평가점수 합계 = Σ(각 행의 이전평가점수)

차이 = 예상평가점수 합계 - 이전평가점수 합계
```

## 팁

- **비중 합계 확인**: 항상 비중 합계가 100%가 되도록 입력하세요
- **멀티라인 설명**: Enter 키로 여러 줄 작성 가능
- **자동 저장 없음**: 작업 후 반드시 수동으로 저장하세요
- **.dat 파일 보안**: 일반 텍스트 편집기로 열람 불가능

## 문제 해결

### 애플리케이션이 시작되지 않는 경우

1. `node build.js` 명령으로 빌드 먼저 실행
2. `dist` 폴더가 생성되었는지 확인
3. `node_modules` 폴더가 있는지 확인 (없으면 `npm install` 실행)

### 저장/불러오기가 작동하지 않는 경우

1. 파일 경로에 특수문자가 없는지 확인
2. 쓰기 권한이 있는 폴더인지 확인
3. `.dat` 확장자로 저장되었는지 확인

## 개발 모드

개발자 도구를 열려면 `src/main/main.ts` 파일에서:

```typescript
process.env.NODE_ENV = 'development';
```

설정 후 재빌드하면 개발자 도구가 자동으로 열립니다.
