# 기여도 평가 점수 프로그램

Electron 기반 데스크톱 애플리케이션으로 기여도 예상 점수를 관리하고 계산하는 프로그램입니다.

## 주요 기능

- **다중 탭 지원**: 여러 개의 탭을 만들어 각각 독립적인 데이터 관리
- **탭 이름 변경**: 탭 이름을 더블클릭하여 자유롭게 변경
- **실시간 계산**: 비중과 점수 입력 시 즉시 평가점수 자동 계산
- **멀티라인 설명**: 여러 줄로 작성 가능한 설명란
- **현재/이전 비교**: 예상평가점수와 이전평가점수를 비교하여 차이 표시
- **안전한 저장**: .dat 파일 형식으로 모든 탭 데이터 저장 (Base64 인코딩)
- **다크 테마**: 눈의 피로를 줄이는 다크 모드 UI

## 설치 및 실행

### 1. 의존성 설치

```powershell
npm install
```

### 2. 개발 모드 실행

**방법 1: 명령어 실행**
```powershell
npm run dev
```

**방법 2: 배치 파일 실행 (Windows)**
```
실행.bat 더블클릭
```

### 3. 배포용 EXE 파일 생성

**방법 1: 명령어 실행**
```powershell
npm run dist
```

**방법 2: 배치 파일 실행 (Windows)**
```
빌드_실행.bat 더블클릭
```

빌드된 실행 파일은 `release` 폴더에 생성됩니다:
- `기여도 평가 점수 프로그램 1.0.0.exe` (포터블 버전 - 설치 불필요)

**포터블 버전 특징:**
- 설치 없이 바로 실행
- USB나 네트워크에서도 실행 가능
- 레지스트리 수정 없음

자세한 빌드 가이드는 `BUILD_GUIDE.md` 파일을 참조하세요.

## 사용 방법

### 탭 관리

1. **탭 추가**: 오른쪽 상단 '+' 버튼으로 새 탭 생성
2. **탭 이름 변경**: 탭 이름을 더블클릭하여 변경
3. **탭 전환**: 탭을 클릭하여 전환
4. **탭 닫기**: 탭 우측 '×' 버튼으로 닫기 (최소 1개 탭 유지)

### 데이터 입력

1. **행 추가**: "+ 행 추가" 버튼으로 새로운 평가 항목 추가
2. **데이터 입력**: 설명, 날짜, 비중, 점수 입력
3. **실시간 계산**: 입력 즉시 예상평가점수 자동 계산
4. **행 삭제**: 체크박스 선택 후 "× 행 삭제" 버튼으로 삭제

### 파일 메뉴

1. **Save (Ctrl+S)**: 현재 파일에 저장 (모든 탭 데이터 포함)
2. **Save As (Ctrl+Shift+S)**: 다른 이름으로 .dat 파일 저장
3. **Load (Ctrl+O)**: 저장된 .dat 파일 불러오기 (모든 탭 복원)

## 계산 공식

- **예상평가점수** = 비중(%) × 점수 / 100
- **예상평가점수 합계** = Σ(비중 × 점수 / 100)
- **이전평가점수** = 이전 비중(%) × 이전 점수 / 100
- **이전평가점수 합계** = Σ(이전 비중 × 이전 점수 / 100)
- **차이** = 예상평가점수 합계 - 이전평가점수 합계
