# 기여도 평가 점수 프로그램

Electron 기반 데스크톱 애플리케이션으로 기여도 예상 점수를 관리하고, 경력 기술서 작성을 위한 PDF 리포트를 생성하는 프로그램입니다.

## 주요 기능

- **다중 탭 지원**: 여러 개의 탭을 만들어 프로젝트/기간별 독립적인 데이터 관리
- **탭 관리**: 탭 이름 변경(더블클릭), 순서 관리 및 삭제
- **실시간 계산**:
  - **Global Averaging**: (모든 탭의 점수 합계) ÷ (탭 개수)로 전체 평균 산출
  - 비중(%)과 점수 입력 시 즉시 반영
- **경력 기술서 PDF 내보내기 (New)**:
  - 작성된 내용을 깔끔한 보고서 양식으로 변환 (A4 최적화)
  - **미리보기 제공**: 저장 전 내용 확인 및 확대/축소(`Ctrl+Wheel`) 지원
  - **자동 기간 산출**: 입력된 날짜 중 최솟값~최댓값을 찾아 총 활동 기간 표시
- **사용자 가이드 (New)**: 탭으로 분리된 도움말 및 업데이트 내역 제공
- **데이터 관리**:
  - 안전한 저장 (.dat 파일, Base64 인코딩)
  - 현재/이전 점수 비교 및 증감 시각화 (▲/▼)
- **다크 테마**: 눈의 피로를 줄이는 다크 모드 UI

## 설치 및 실행

### 1. 의존성 설치

```powershell
npm install
````

### 2. 개발 모드 실행

**방법 1: 명령어 실행**

```powershell
npm run dev
```

**방법 2: 배치 파일 실행 (Windows)**

```
실행.bat 더블클릭
```

### 3\. 배포용 EXE 파일 생성

**방법 1: 명령어 실행**


```

```

빌드된 실행 파일은 `release` 폴더에 생성됩니다:

  - `기여도 평가 점수 프로그램 2.1.0.exe` (포터블 버전)

## 사용 방법

### 기본 사용

1.  **데이터 입력**: 행 추가 후 설명, 날짜, 비중, 점수 입력
2.  **탭 활용**: 프로젝트나 기간 단위로 탭을 나누어 관리
3.  **저장**: `Ctrl+S`로 수시로 저장

### PDF 내보내기 (경력 기술서)

1.  메뉴의 `File` \> `Export to PDF` 클릭 또는 `Ctrl+P` 입력
2.  **미리보기 창**이 뜨면 내용을 확인
      - **확대**: `Ctrl` + `마우스 휠 아래로` (또는 ＋ 버튼)
      - **축소**: `Ctrl` + `마우스 휠 위로` (또는 － 버튼)
3.  `저장하기` 버튼을 눌러 PDF 파일로 저장

## 계산 공식

  - **예상평가점수** = 비중(%) × 점수 / 100
  - **탭별 점수 합계** = Σ(해당 탭의 예상평가점수)
  - **최종 평균 점수** = (모든 탭의 점수 합계 총합) / (탭 개수)

  - **Electron**: 데스크톱 앱 프레임워크 (v28)
  - **TypeScript**: 메인 프로세스 타입 안정성 확보
  - **HTML/CSS/JS**: 렌더러 프로세스 UI 구현
  - **Node.js**: 파일 시스템 접근 및 PDF 생성
