# 키보드 단축키

## File 메뉴

| 단축키 | 기능 | 설명 |
|--------|------|------|
| `Ctrl + S` | Save | 현재 파일에 저장 |
| `Ctrl + Shift + S` | Save As | 다른 이름으로 저장 |
| `Ctrl + O` | Load | 파일 불러오기 |
| `Alt + F4` | Exit | 프로그램 종료 |

## Edit 메뉴

| 단축키 | 기능 |
|--------|------|
| `Ctrl + Z` | Undo (실행 취소) |
| `Ctrl + Y` | Redo (다시 실행) |
| `Ctrl + X` | Cut (잘라내기) |
| `Ctrl + C` | Copy (복사) |
| `Ctrl + V` | Paste (붙여넣기) |
| `Ctrl + A` | Select All (전체 선택) |

## View 메뉴

| 단축키 | 기능 |
|--------|------|
| `Ctrl + R` | Reload (새로고침) |
| `Ctrl + Shift + I` | Toggle Developer Tools (개발자 도구) |
| `Ctrl + 0` | Reset Zoom (확대/축소 초기화) |
| `Ctrl + +` | Zoom In (확대) |
| `Ctrl + -` | Zoom Out (축소) |

## 입력 필드

| 키 | 기능 |
|-----|------|
| `Enter` | 날짜 필드에서 포맷 적용 |
| `Tab` | 다음 필드로 이동 |

## 팁

- **빠른 저장**: 작업 중 자주 `Ctrl + S`를 눌러 저장하세요
- **날짜 입력**: 날짜 필드에서 `20240101` 입력 후 `Enter`를 누르면 `2024.01.01`로 자동 포맷됩니다
